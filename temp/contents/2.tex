\section{有序列表、无序列表和描述列表}

\subsection{有序列表}

中华人民共和国由下列国家机构组成：

\begin{Enumerate}
\item {中国共产党}
\item {全国人民代表大会}
\item {国务院}
\item {中央政治协商会议}
\item {中央军事委员会}
\end{Enumerate}

\subsection{无序列表}

中华人民共和国由下列国家机构组成：

\begin{Itemize}
\item {中国共产党}
\item {全国人民代表大会}
\item {国务院}
\item {中央政治协商会议}
\item {中央军事委员会}
\end{Itemize}

\subsection{描述列表}

中华人民共和国由下列国家机构组成：

\begin{Description}[12em]
\item[中国共产党] {统御全局的领导核心。}
\item[全国人民代表大会] {负责立法工作。}
\item[国务院] {负责全国行政事务。}
\item[中央政治协商会议] {负责建言献策。}
\item[中央军事委员会] {负责保卫国家安全和领土完整。}
\end{Description}
